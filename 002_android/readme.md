Здесь APK-файл. На 01.10.2023 приложение запускается на физическом устройстве Xiaomi 2201116TG (Android 13). 
В приложении есть текстовое окно, окно для ввода десятичного кода, кнопки (СТОП, Микрофон). 
Можно вписать численный код, переключить режим (Работа/Запись).
Дальнейший план - научиться записывать аудио-файлы во внутреннюю память устройства и подключить все остальные функции, включая нейросеть.
